# 🛠️ Utils Package – CineVault

The **utils** package contains helper classes that provide common functionalities used across the application.

## 📁 Classes

* **Validator.java**
  Handles input validation such as checking empty fields, and basic constraints.

* **SessionManager.java**
  Manages the current logged-in user session throughout the application.

## 🎯 Purpose

* Provides reusable utility functions
* Reduces code duplication
* Supports controllers and views with common operations

## ⚠️ Notes

* Does not contain business logic or database code
* Methods are usually **static** for easy access
* Keeps helper logic separate from main application layers
