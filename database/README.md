# 🗄️ Database – CineVault

This folder contains the database schema for the CineVault project.

## 📄 Files

* **cinevault.sql**
  Contains SQL queries to create the database and required tables.

## 🧱 Tables

* **users** – Stores user information
* **movies** – Stores movie details
* **ratings** – Stores user ratings for movies

## 🎯 Purpose

* Defines the structure of the database
* Supports CRUD operations in the application
* Maintains relationships between users, movies, and ratings

## ⚠️ Note

This is the initial schema and may be updated as the project evolves.
