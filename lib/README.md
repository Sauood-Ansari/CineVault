# 📚 Lib – CineVault

This folder contains external libraries required for the project.

## 📦 Dependencies

* **MySQL Connector/J**
  JDBC driver used to connect the Java application with the MySQL database.

## 🎯 Purpose

* Enables database connectivity using JDBC
* Provides required drivers for executing SQL queries from Java

## ⚠️ Note

Make sure this JAR file is added to the project build path before running the application.
